# 📚 課程總覽與說明

本目錄包含企業級RAG課程的整體介紹和完成總結文檔。

## 📋 文檔說明

### `ENTERPRISE_RAG_COURSE_OVERVIEW.md` - 課程大綱總覽
- **目的**: 課程整體介紹和學習路徑
- **受眾**: 新學員、課程規劃者
- **內容**: 學習目標、技術棧、12週課程結構
- **使用時機**: 課程開始前的準備階段

### `COURSE_COMPLETION_SUMMARY.md` - 課程完成總結
- **目的**: 展示課程內容完成度和學習成果
- **受眾**: 完成學習的學員、教學評估者
- **內容**: 章節完成度、學習時數、認證體系
- **使用時機**: 課程結束後的成果驗收

## 🎯 文檔使用指南

### 新學員入門
1. 先閱讀 `ENTERPRISE_RAG_COURSE_OVERVIEW.md` 了解整體課程
2. 查看學習目標和技術要求
3. 確認時間安排（12週密集 或 6個月兼職）

### 課程完成驗收
1. 閱讀 `COURSE_COMPLETION_SUMMARY.md` 檢查學習進度
2. 對照完成的章節和實作項目
3. 評估認證等級和技能水平
